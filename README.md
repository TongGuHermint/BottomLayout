# BottomLayout
底部导航栏控件
## Gradel
 ### Add it in your root build.gradle at the end of repositories:
  ```
  allprojects {
	  repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
	}
   ```
  ### Add the dependency
  ```
  dependencies {
      implementation 'com.github.TongGuHermint:BottomLayout:v1.0.1'
	}
```
